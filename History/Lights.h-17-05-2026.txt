Lights.h — History 17-05-2026
Conditionally safeguarded DirectX namespace.

Changes:
- 'using namespace DirectX;' now guarded:
  #if defined(__USE_DIRECTX_11__) || defined(__USE_DIRECTX_12__)
  using namespace DirectX;
  #endif
- LightStruct fields using XMFLOAT3 now resolve via Includes.h aliases (Vector3) on non-DX builds.
- No structural changes to LightStruct layout — binary layout is identical (Vector3 has same
  memory footprint as XMFLOAT3).
