ConsoleWindow.h - Created 16-05-2026
=====================================
New file. Declares the ConsoleWindow class:
  - 50-line circular buffer (CONSOLE_MAX_LINES)
  - Max 10 visible lines (CONSOLE_VISIBLE_LINES)
  - Vertical scrollbar on the right edge (CONSOLE_SCROLLBAR_W = 10px)
  - Titlebar height CONSOLE_TITLEBAR_H = 20px
  - CONSOLE_PADDING = 4px inner padding
  - bIsVisible flag toggled by F8
  - Thread-safe AddLine() / Clear() via std::mutex
  - Render(Renderer*, int w, int h) draws the full window using DrawRectangle + DrawMyText

Debug-console integration update (same day):
  - Added ConsoleLineColor enum { Normal, Warning, Error }
  - Added ConsoleLine struct { std::wstring text; ConsoleLineColor type }
  - m_buffer changed from std::deque<std::wstring> to std::deque<ConsoleLine>
  - AddLine() gains optional ConsoleLineColor type parameter (default Normal)

Scrollbar fix + resize update (same day):
  - CONSOLE_VISIBLE_LINES raised from 10 to 14 (taller window)
  - Added Scroll(int delta) method declaration

Mouse input update (same day):
  - Added HandleMouseWheel(int delta) and HandleMouseClick(float x, float y) declarations
  - Added private cached scrollbar rect members: m_sbX, m_sbY, m_sbH (set each frame in Render)
  - CONSOLE_MAX_LINES raised from 50 to 100
