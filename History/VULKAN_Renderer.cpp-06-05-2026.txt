VULKAN_Renderer.cpp - 06-05-2026
==================================
NEW FILE

Full VulkanRenderer implementation — Vulkan 1.2 equivalent of DX11Renderer.cpp.

Key additions:
- CreateInstance() with platform-specific extensions (Win32/XCB/Android surface)
- SetupDebugMessenger() with DebugCallback()
- CreateSurface() dispatching to vkCreateWin32SurfaceKHR / vkCreateXcbSurfaceKHR /
  vkCreateAndroidSurfaceKHR per platform
- PickPhysicalDevice() with scoring (discrete > integrated)
- CreateLogicalDevice() with graphics + present queues
- CreateSwapChain() with format/mode/extent selection
- CreateRenderPass() with color + depth attachments
- CreateGraphicsPipelines(): 2D alpha-blend quad pipeline + 3D depth-tested pipeline;
  inline GLSL compiled at runtime via shaderc (__has_include guard)
- CreateOverlayResources(): Windows = WIC bitmap + D2D render target;
  Linux/Android = stubs
- UploadOverlayToVulkan(): WIC lock -> staging map -> vkCmdCopyBufferToImage with
  layout transitions
- Full texture helpers: CreateTextureFromRGBA, DestroyVulkanTexture, CreateBuffer,
  CopyBuffer, CreateImage, CreateImageView, TransitionImageLayout
- LoadTexture(): WIC (Windows) / stb_image stub (Linux/Android)
- All Draw* / Blit2D* / CalculateText* methods routing through D2D (Windows) or stubs
- VKDeltaTimeSmoothing implementation
- BeginFrame() / EndFrame() / RecreateSwapChain() / OnWindowResize()
