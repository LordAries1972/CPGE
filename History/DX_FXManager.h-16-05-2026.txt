WarpDotTunnel enhancements — 16/05/2026

TunnelRing struct:
- Added: int colorStep = 0  (index into the dark-to-white gray ramp; set at init time, cycles via ringIndex % kGraySteps)

WarpTunnelData struct:
- Added: float sideWaveTime = 0.0f  (accumulates each frame; drives furthest-ring left/right sine oscillation)
- Added: static constexpr float kSideWaveRadius = 30.0f  (peak ±X swing in world units)
- Added: static constexpr float kSideWaveSpeed  = 0.5f   (oscillation frequency in rad/s)
- Added: static constexpr int   kGraySteps      = 8       (number of shades in the dark-to-white ramp)
