Debug.cpp - Updated 16-05-2026
================================
Debug-console integration update:
  - Added #ifdef _DEBUG include of ConsoleWindow.h + extern ConsoleWindow consoleWindow
  - logLevelMessage(): after OutputDebugStringW, forwards tagged message to consoleWindow
    using ConsoleLineColor mapped from LogLevel:
      LOG_WARNING               → ConsoleLineColor::Warning  (yellow)
      LOG_ERROR / LOG_CRITICAL  → ConsoleLineColor::Error    (orange)
      all others                → ConsoleLineColor::Normal   (white)
    This automatically covers logDebugMessage() and LOG_IF_FAILED() which both
    route through logLevelMessage().
  - Log()         → consoleWindow.AddLine(L"[INFO]: " + msg,     Normal)
  - LogWarning()  → consoleWindow.AddLine(L"[WARNING]: " + msg,  Warning)
  - LogError()    → consoleWindow.AddLine(L"[ERROR]: " + msg,    Error)
  - DebugLog()    → consoleWindow.AddLine(L"[INFO]: " + msg,     Normal)
  - LogFunction() → consoleWindow.AddLine(fullMessage,            Normal)
  All forwarding is inside #ifdef _DEBUG blocks; Release builds are unchanged.
