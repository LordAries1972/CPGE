Includes.h - History - 18-05-2026
==================================

NEW: #define GAME_NAME "CPGE"

Added a game identity define at the top of the file (after #pragma once)
that sets the base name used in the output executable:

  Windows  : DX{GAME_NAME}.exe / OpenGL{GAME_NAME}.exe / Vulkan{GAME_NAME}.exe
  macOS    : DX{GAME_NAME}.app / OpenGL{GAME_NAME}.app / Vulkan{GAME_NAME}.app
  Linux    : OpenGL{GAME_NAME} / Vulkan{GAME_NAME}

Default value: "CPGE"

NOTE: Must be kept in sync with:
  - <GameName> in CrossPlatformGameEngine.vcxproj
  - GAME_NAME cmake cache variable in CMakeLists.txt
