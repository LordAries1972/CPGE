VULKAN_IOStreamThread.cpp - 06-05-2026
========================================
NEW FILE

VulkanRenderer::LoaderTaskThread() implementation — Vulkan port of IOStreamDX11Thread.cpp.

Key additions:
- Guarded by #if defined(__USE_VULKAN__)
- std::mutex VulkanRenderer::s_loaderMutex definition
- Same scene-switch structure as DX11 version:
    SCENE_INTRO    — LoadAllKnownTextures()
    SCENE_GAMETITLE — textures + light + GLTF parse + starfield + FadeToImage (first load);
                      camera/GUI resize only (wasResizing path)
    SCENE_LOAD_MP3 — Windows only: CoInitialize + MP3 player; CoUninitialize cleanup
    SCENE_GAMEPLAY — textures + light + GLTF parse + Load_Music (first load only)
    SCENE_GAMEOVER — log only
- SecureZeroMemory for LightStruct on PLATFORM_WINDOWS; zero-init on Linux/Android
- sysUtils.ProcessMessages() calls guarded by PLATFORM_WINDOWS
- CoInitializeEx / CoUninitialize guarded by PLATFORM_WINDOWS && __USE_MP3PLAYER__
- Extern references: VKFXManager fxManager (not FXManager); ShaderManager and
  SystemUtils conditionally declared for PLATFORM_WINDOWS only
