Added bool isBorderless field to WindowMetrics struct (after the existing isFullScreen field).
Initialised to false in the WindowMetrics default constructor initialiser list.

Purpose: main.cpp display-mode setup sets this flag to true when borderless windowed mode is active
so that WindowProc and any other runtime code can distinguish borderless from ordinary windowed without
querying the window style each frame.
