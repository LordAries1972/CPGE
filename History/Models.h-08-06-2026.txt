// Changes:
//
//   Vertex struct (DX11/DX12):
//     Added: float tangentW = 1.0f
//     Previously tangent was float3 (xyz only); tangentW carries the GLTF handedness
//     sign (+1 or -1) for correct bitangent computation on mirrored UV islands.
//     Vertex size: 44 -> 48 bytes.
//     NOTE: existing cache.dat files with 44-byte vertices must be regenerated.
//
//   Vertex struct (OpenGL/Vulkan):
//     Changed: float tangent[3] -> float tangent[4]
//     tangent[3] carries the handedness sign matching GLTF VEC4 TANGENT attribute.
//
//   Material struct:
//     Added: std::string glossMapPath
//     Added: std::string emissiveMapPath
//     Added: std::shared_ptr<Texture> glossMap
