VULKAN_Renderer.h - 06-05-2026
===============================
NEW FILE

Full VulkanRenderer class declaration — Vulkan 1.2 equivalent of DX11Renderer.h.

Key additions:
- Platform guards: PLATFORM_WINDOWS (Win32 surface + D2D/DWrite overlay + DirectXMath),
  PLATFORM_LINUX (XCB surface + GLM), PLATFORM_ANDROID (ANativeWindow + GLM)
- Structs: QueueFamilyIndices, SwapChainSupportDetails, VulkanTexture, VkVertex2D/3D,
  VulkanFrameData
- VKDeltaTimeSmoothing class (port of DX11 DeltaTimeSmoothing)
- VulkanRenderer : public Renderer with all pure-virtual overrides
- VK_MAX_FRAMES_IN_FLIGHT = 2 frames-in-flight pattern
- Static s_loaderMutex for loader thread (LoaderTaskThread defined in VULKAN_IOStreamThread.cpp)
- Private members: instance, physical/logical device, queues, swapchain, render pass,
  framebuffers, depth image, command pool, frame sync objects, 2D/3D pipelines,
  texture arrays, overlay image + staging buffer
- Windows-only: WIC factory/bitmap, D2D factory/render target, DWrite factory, text format cache
- GetCurrentCommandBuffer(), GetRenderPass(), GetVkDevice() accessors for FXManager use
