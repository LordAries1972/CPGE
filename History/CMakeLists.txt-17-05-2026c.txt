CMakeLists.txt — History 17-05-2026c
OpenGL/Vulkan shader compilation steps added; OpenGLModels/VulkanModels added to SOURCES.

Changes:
- OpenGLModels.cpp and VulkanModels.cpp added to SOURCES list.
- OpenGL GLSL shader copy step (POST_BUILD):
  copy_glsl_shader(ModelVertex.glsl) and copy_glsl_shader(ModelPixel.glsl)
  OpenGL loads GLSL text at runtime from the Windows SDK opengl32.lib — no compilation step needed.
  Status message: "[Shaders] OpenGL: GLSL source files copied to output dir (loaded at runtime)"
- Vulkan SPIR-V shader compilation (PRE_BUILD):
  Uses glslangValidator from Vulkan SDK ($ENV{VULKAN_SDK}/bin or Bin).
  compile_vulkan_shader(ModelVertex.vert ModelVertex.vert.spv)
  compile_vulkan_shader(ModelPixel.frag ModelPixel.frag.spv)
  Fallback: if glslangValidator not found, copies raw .vert/.frag for shaderc runtime compilation.
  Status messages identifying which tool compiles which shaders.
- Vulkan SDK auto-detection block: find_package(Vulkan) with drive-root fallback scan.
- FXC HLSL compiler detection and HLSL shader pre-compilation already present (unchanged).
